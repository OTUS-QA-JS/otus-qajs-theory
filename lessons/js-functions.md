## Что такое JavaScript
**JavaScript** - это высокоуровневый язык программирования с динамической типизацией. Динамическая типизация означает, что тип данных переменной определяется в момент присваивания значения и может изменяться во время выполнения программы.

## Функции

### Объявление функций
Функция - это многократно используемый блок кода, который выполняет определенную задачу. Функции объявляются с помощью ключевого слова `function`, за которым следует имя и круглые скобки.

```javascript
// Объявление функции
function funcName() {
  // код функции
}
```

### Параметры и аргументы
Функции могут принимать **параметры** (переменные, перечисленные в объявлении функции) и получать **аргументы** (значения, передаваемые в функцию при ее вызове).

```javascript
function hello(name) { // name - это параметр
  console.log("Привет, " + name);
}

hello('Student'); // 'Student' - это аргумент
```

### Возврат значения
По умолчанию функции возвращают `undefined`. Для возврата результата используется ключевое слово `return`, которое также прекращает выполнение функции.

```javascript
const check = () => {
  console.log('Этот код выполнится');
  return 'результат выполнения';
  console.log('Этот код уже не выполнится');
}

console.log(check());
// Вывод:
// Этот код выполнится
// результат выполнения
```

### Стрелочные функции
Это сокращенный синтаксис для объявления функций. Они обеспечивают более краткую и выразительную форму записи.

```javascript
// Обычная функция
function add(a, b) {
  return a + b;
}

// Стрелочная функция
const addArrow = (a, b) => {
  return a + b;
};

// Или еще короче (неявный возврат)
const addArrowShort = (a, b) => a + b;
```

## Типы данных
Основные типы данных в JavaScript, необходимые для начала работы:

### string (строки)
Любые текстовые данные. Строки можно объявлять используя двойные (`"`), одинарные (`'`) или обратные кавычки (`` ` ``).

**Экранирование** - это способ включить в строку специальные символы. Если внутри строки нужно использовать тот же тип кавычек, что и для ее создания, их необходимо экранировать обратным слэшем `\`.

```javascript
console.log('Символ \'одинарной кавычки\' внутри строки.'); // Символ 'одинарной кавычки' внутри строки.
console.log("Символ \"двойной кавычки\" внутри строки."); // Символ "двойной кавычки" внутри строки.
```

**Шаблонные строки** (объявленные через обратные кавычки) позволяют встраивать переменные и выражения непосредственно в строку.

```javascript
const a = 10;
// Подстановка переменной в обычную строку
console.log('Переменная a равна ' + a + '.'); // Переменная a равна 10.
// В шаблонную строку
console.log(`Переменная a равна ${a}.`); // Переменная a равна 10.
```

### number (числа)
Включает в себя как целые числа, так и числа с плавающей точкой. Специальные числовые значения: `NaN` (Not-a-Number), `Infinity` и `-Infinity`.

**Математические операторы:**
- `+`, `-`, `*`, `/` — сложение, вычитание, умножение, деление.
- `%` — остаток от деления (`11 % 2` вернет `1`).
- `**` — возведение в степень (`2 ** 3` вернет `8`).

### boolean (логический тип)
Имеет всего два значения: `true` (истина) и `false` (ложь).

**Операторы сравнения:**
- `==` — нестрогое равенство (с приведением типов).
- `===` — строгое равенство (без приведения типов, сравнивает и значения, и типы).
- `!=`, `!==` — нестрогое и строгое неравенство.
- `>`, `<`, `>=`, `<=` — больше, меньше, больше или равно, меньше или равно.

**Логические операторы:**
- `!` — логическое "НЕ" (отрицание). `!true` вернет `false`.

### null и undefined
- `undefined` — означает, что значение не было присвоено. Переменные, объявленные без значения, по умолчанию получают `undefined`.
- `null` — означает "пустое" или "неизвестное" значение. Используется, чтобы явно указать на отсутствие объекта.

## Переменные
Переменные используются для хранения данных.

- `let` — объявляет переменную, значение которой можно изменять (переприсваивать).
- `const` — объявляет константу, значение которой нельзя изменять после инициализации.

```javascript
let user;
user = 'Иван'; // Присвоили значение
user = 'Петр'; // Переприсвоили

const baseUrl = 'http://site.com';
// baseUrl = ''; // Ошибка: Assignment to constant variable.
```

## Модули
JavaScript позволяет разделять код на несколько файлов (модулей) для лучшей организации.

### Экспорт и импорт
Функции, переменные или классы можно **экспортировать** из одного модуля и **импортировать** в другой.

```javascript
// В файле module.js
export function funcName() {
  // код функции
}

// В файле main.js
import { funcName } from "./module.js";
```

### Типы модулей
- **ES Modules (ESM)**: Стандартный формат модулей в современном JavaScript. Использует синтаксис `import`/`export`.
- **CommonJS (CJS)**: Исторически использовался в Node.js. Использует `require()` для импорта и `module.exports` для экспорта.

Тип используемых модулей в проекте Node.js настраивается в файле `package.json`.

## Инструменты
- **Babel**: Транспилятор, который преобразует современный JavaScript в обратно совместимый код для старых браузеров или сред.
- **ESLint**: Линтер, анализирует код на предмет стилистических и синтаксических ошибок.
- **Prettier**: Инструмент для автоматического форматирования кода, обеспечивает единый стиль.
- **EditorConfig**: Помогает поддерживать единые настройки редактора (отступы, кодировка) в команде.
- **npm-run-all**: Позволяет запускать несколько npm-скриптов параллельно или последовательно.

### Пример скриптов в `package.json`
```json
{
  "scripts": {
    "start": "babel-node src/main.js",
    "format": "prettier . --check",
    "format:fix": "prettier . --write",
    "lint": "eslint .",
    "lint:fix": "eslint . --fix",
    "codestyle": "run-p lint format",
    "codestyle:fix": "run-s lint:fix format:fix",
    "build": "babel src --out-dir build --source-maps inline"
  }
}
```
