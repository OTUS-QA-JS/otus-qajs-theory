# Охота за локаторами: практика поиска элементов на странице

## Цель
Практика в поиске локаторов на странице

## Задание
1. откройте страницу Login на сайте https://rwa-194.87.102.103.sslip.io/login или https://demo.realworld.io/#/login
2. напишите в ДЗ, какой сайт вы решили выбрать
3. найдите локатор формы авторизации
4. найдите локатор кнопки `Sign in`
5. найдите локатор поля ввода `Password`

Можно использовать `css`-селекторы или `xpath`. Если к этому вебинару, вы знакомы с одним из тестовых фреймворков (playwright / cypress / ...) для UI тестов, можете прислать локаторы, с использованием возможностей этих фреймворков.

---
## Критерии оценивания
* найдены 4 локатора
