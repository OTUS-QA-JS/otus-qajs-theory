# GIT

## Цель
Продемонстрируйте базовый уровень владения git и github.com

## Задание
1. cоздать репозиторий на GitHub
2. создать ветку master
3. создать ветку develop
4. в ветке develop создать файл README.md
5. в файле README.md написать: Создано в рамках курса OTUS JavaScript QA Engineer
6. зафиксировать изменения (сделать коммит)
7. отправить изменения в удалённый репозиторий (учтите, что вам нужно отправить в удалённый репозиторий две ветки, master и develop)
8. создать pull request на github из ветки develop в ветку master
9. отправить ссылку на pul request в чат проверки ДЗ
